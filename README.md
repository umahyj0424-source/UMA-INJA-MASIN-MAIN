# UMA 인자작 & 마신표 추천기 - 수동 편집형

이 버전은 정적 사이트입니다. 서버 DB 없이 브라우저에서 마신표, 스킬 DB, 스킬 목록을 직접 편집할 수 있습니다.

## 배포

Netlify에 이 폴더 전체를 업로드하거나, GitHub 저장소에 올린 뒤 Netlify와 연결합니다.

필수 파일 구조:

```txt
index.html
style.css
app.js
data/site-data.json
```

## 수정 데이터 저장 방식

사이트에서 수정한 데이터는 먼저 브라우저 localStorage에 저장됩니다.
다른 사람에게 같은 데이터를 공유하거나 영구 배포하려면:

1. 사이트의 `데이터 관리` 탭에서 `site-data.json 다운로드`
2. GitHub 저장소의 `data/site-data.json` 파일 교체
3. Netlify 재배포

## 마신표 붙여넣기

`마신표` 탭에서 코스와 각질을 고르고, 엑셀에서 아래 열을 복사해 붙여넣습니다.

```txt
스킬명	마신이득	마신/Pt	발동률	유효율
```

첨부한 `2026 6월 단거리 말오스 마신표의 사본.xlsx`처럼 도주/선행/선입/추입 시트의 A:E 범위를 그대로 붙여넣으면 됩니다.

## 스킬 DB 붙여넣기

`스킬 DB` 탭에서 아래 구조로 붙여넣습니다.

```txt
스킬명	캐릭터	캐릭터 (이벤트 획득)	서포트 카드 (힌트 획득)	서포트 카드 (이벤트 획득)	상태
```

여러 캐릭터/카드는 쉼표로 구분합니다.

## 스킬 목록

`스킬 목록` 탭에서 한 줄에 하나씩 스킬명을 입력합니다. 마신표/스킬 DB의 스킬명을 합쳐 자동완성 목록에 반영할 수도 있습니다.
