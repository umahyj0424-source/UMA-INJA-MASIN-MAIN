# utools 마신표 URL 목록
# 한 줄에 하나의 URL을 넣으면 됩니다.
# 빈 줄과 #으로 시작하는 줄은 무시됩니다.
#
# 가장 추천하는 기본 형식: 코스의 /effects 주소만 넣기
# 아래처럼 넣으면 자동으로 runner/leader/betweener/chaser 4개 URL을 만들어 긁습니다.
# https://xn--gck1f423k.xn--1bvt37a.tools/race/courses/10501/effects
#
# 즉 아래 1줄은 내부적으로 아래 4줄과 같습니다.
# https://.../effects/runner     = 도주
# https://.../effects/leader     = 선행
# https://.../effects/betweener  = 선입
# https://.../effects/chaser     = 추입
#
# 코스명을 직접 붙이고 싶으면:
# 6월 단거리 10501|https://xn--gck1f423k.xn--1bvt37a.tools/race/courses/10501/effects
#
# 특정 각질만 직접 지정하고 싶으면:
# 6월 단거리 10501|선행|https://xn--gck1f423k.xn--1bvt37a.tools/race/courses/10501/effects
# 6월 단거리 10501|선행|https://xn--gck1f423k.xn--1bvt37a.tools/race/courses/10501/effects/leader
